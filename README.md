Get WKT (QGIS 3.0)
===========

QGIS plugin that allows you to view the WKT of a geometry.

![](screenshot.png)

Based on the QGIS 2.0 Plugin of the same name by Hugo Ledoux https://github.com/hugoledoux/QGIS-getWKT. Migrated to QGIS 3.0 by Paul Skeen.
